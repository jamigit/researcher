# .cursor Directory

This directory contains **AI-friendly documentation** in MDC (Markdown Cursor) format.

## Purpose

The `.cursor/` directory stores documentation optimized for AI assistants and Cursor IDE features, including:

- **Architecture documentation** (`architecture.mdc`) - System design, data flow, and technical patterns
- **Instructions** (`instructions.mdc`) - Setup, development, and operational procedures
- **Rules** (`rules/`) - Code standards, patterns, and guidelines for different layers
- **Agents** (`agents/`) - Specialized AI agent configurations for specific tasks
- **Commands** (`commands.json`) - Custom Cursor commands and shortcuts

## Format

Files use `.mdc` extension (Markdown Cursor format) which is markdown with additional metadata and structure optimized for AI consumption.

## Relationship to `/docs`

- `.cursor/` → AI-optimized, structured documentation
- `/docs/` → General project documentation for human readers
